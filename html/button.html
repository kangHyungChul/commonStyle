PC Button

가장 기본적인 UI 요소로 화면을 전환하거나 새로운 정보를 호출
.c-button클래스를 기본으로 사용

Size

  • .c-button--#{size} modifier을 추가하여 사용
  • Default사이즈는 md(40)로 별도의 modifier없이 사용
Property Sample Code
xl (64) 
<button type="button" class="c-button c-button--xl">버튼</button>
lg (48) 
<button type="button" class="c-button c-button--lg">버튼</button>
md (40) / 기본 
<button type="button" class="c-button">버튼</button>
sm (32) 
<button type="button" class="c-button c-button--sm">버튼</button>
xs (28) 
<button type="button" class="c-button c-button--xs">버튼</button>
xxs (24) 
<button type="button" class="c-button c-button--xxs">버튼</button>

Style

  • .c-button--#{style} modifier을 추가하여 사용
  • Default스타일은 별도 modifier없이 사용
  • disabled상태는 disabled속성을 추가
Property Sample Code
Default 
<button type="button" class="c-button">Default</button>
primary 
<button type="button" class="c-button c-button--primary">Primary</button>
secondary 
<button type="button" class="c-button c-button--secondary">Secondary</button>
round 
<button type="button" class="c-button c-button--round">Round/md</button>

Text + Icon

  • 버튼 내에 텍스트와 아이콘의 조합일 경우 사용
  • 버튼 내부의 텍스트 앞이나 뒤에 i.#{icon}을 추가하여 사용
Property Sample Code
Left Icon 
<button type="button" class="c-button">
    <i class="arrowleft"></i><span>버튼</span>
</button>
Right Icon 
<button type="button" class="c-button">
    <span>버튼</span><i class="arrowright"></i>
</button>
Left Icon 
<button type="button" class="c-button c-button--primary">
    <i class="arrowleft"></i><span>버튼</span>
</button>
Left Icon 
<button type="button" class="c-button c-button--secondary">
    <i class="arrowleft"></i><span>버튼</span>
</button>

Only Icon

  • .c-button--icon modifier추가하여 사용
  • 완전 원형 버튼은 .c-button--circle modifier추가하여 사용
  • 아이콘은 버튼 내부에i.#{icon}을 추가
  • 아이콘 뒤에 span.hide로 대체텍스트를 포함시켜야 함
Property Sample Code
Icon 단독 
<button type="button" class="c-button c-button--icon">
    <i class="arrowleft"></i><span class="hide">아이콘단독대체텍스트</span>
</button>
Icon 단독 아이콘단독대체텍스트 아이콘단독대체텍스트 아이콘단독대체텍스트 아이콘단독대체텍스트 아이콘단독대체텍스트 아이콘단독대체텍스트 
<a href="#" class="c-button c-button--icon">
    <i class="arrowleft"></i><span class="hide">아이콘단독대체텍스트</span>
</a>
Icon 단독 
<button type="button" class="c-button c-button--icon c-button--primary">
    <i class="arrowleft"></i><span class="hide">아이콘단독대체텍스트</span>
</button>
Icon 단독 
<button type="button" class="c-button c-button--icon c-button--secondary">
    <i class="arrowleft"></i><span class="hide">아이콘단독대체텍스트</span>
</button>
Icon 단독 / 원형
