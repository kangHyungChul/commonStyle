PC Popover

버튼 또는 아이콘 요소를 클릭했을 때 주위에 작게 뜨는 요소로 정보에 대해 단순하고 빠른 전달이 필요할 경우 사용

Position

  • .c-popup--pos-#{position}형태의 modifier로 화살표의 위치를 지정하여 사용 (필수)
  • #{position}은 각각 상(t)/하(b)/좌(l)/우(r) + 상(t)/하(b)/좌(l)/우(r) 형태로 조합
  • example) 화살표가 우측하단일 경우 --rb / 화살표가 하단좌측일 경우 --bl
  • popover의 화살표 위치는 위 modifier로, 자체의 position은 디자인페이지에서 직접 지정한다
